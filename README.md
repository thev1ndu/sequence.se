# Blog

A blog built with Next.js, featuring MDX support for rich content.

## Features

- MDX blog posts with frontmatter
- Table of contents
- Syntax highlighting
- Responsive design
- Dark mode support

## Getting Started

```bash
npm install
npm run dev
```

## Blog Posts

Blog posts are located in `src/components/blog/content/` as MDX files.
