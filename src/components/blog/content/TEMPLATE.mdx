## Frontmatter Reference

Here's what each frontmatter field does:

- **title**: The main title of your blog post (required)
- **description**: SEO description and preview text (required)
- **image**: URL to the featured/social image (optional, recommended: 1200x630px)
- **pinned**: Set to `true` to pin the post to the top of the blog list (optional, default: false)
- **new**: Set to `true` to show a "New" badge (optional, default: false)
- **category**: Category identifier for filtering (optional, default: "blog")
- **icon**: Icon name for visual representation (optional)
- **createdAt**: Publication date in format: YYYY-MM-DD or "MMM D, YYYY" (required)
- **updatedAt**: Last update date in format: YYYY-MM-DD or "MMM D, YYYY" (optional, defaults to createdAt)

## Tips for Writing

1. **Start with a hook**: Your first paragraph should grab attention
2. **Use clear headings**: Make your content scannable
3. **Include visuals**: Images break up text and add interest
4. **Write in your voice**: Be authentic and conversational
5. **Edit thoroughly**: Check for typos and clarity
6. **Add value**: Make sure readers learn something or feel something
